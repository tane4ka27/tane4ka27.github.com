3.5.1. Силовой расчет механизмов ножа

Работа сил резания. Она рассчитывается по формуле

Fрез = kзап pL,

где    L – ширина стопы, см;

р – погонное усилие резания, Н/см (для бумаг рекомендуется среднее значение, равное 200 Н/см);

kзап – коэффициент запаса, учитывающий возможное затупление ножа.

Для определения максимальной силы резания  принимают kзап = 1,5.

Fрез = kзап pL = 1,5∙200∙160 = 48000 Н

Работа резания (работа, расходуемая на преодоление ножом всех сил сопротивления стопы листов) является основной частью всей работы, которую выполняет резальная машина. Если известна кривая зависимости

Fрез = f(S) (см. рис. 1.5, в), то работа резания

где    Sнач и  Sкон – граничные значения пути ножа.

Для практических инженерных расчетов условно принимают усилие резания Fрез  постоянным, тогда

Арез = Fрез ∙S,

где    S – длина пути ножа во время резания, которая определяется высотой стопы h и усредненным углом траектории ножа Θ, принимаем Θ = 45°:

S = h/sinΘ = 140/sin 45° = 140/ 0,7071 = 197,99 мм

Арез = Fрез ∙S = 48000∙197,99∙10–3 = 9503,52 Дж,

Средняя мощность, расходуемая на резание за цикл,

Ррез = Арез / tц / ηмех = 9503,52 /1,5/0,98 = 6464,98 Вт,

где    tц = 60/n – время одного кинематического цикла машины;

          п – количество циклов резания в минуту при непрерывной работе (40…50 цикл/мин), примем п = 40, тогда tц = 60/n = 60/40 = 1,5 с;

ηмех – КПД механизма ножа, примем ηмех = 0,98.

Фактическое время резания tц будет определяться углом поворота кривошипа φрез, соответствующим периоду разрезки стопы, т. е.

tц = φрез / ωвала кр,

где    φрез = φ2 – φ1,  здесь φ2 и φ1 — углы, соответствующие началу врезания  ножа в стопу и марзан, радиан (см. рис. 1.13, а);

         ωвала кр – угловая скорость приводного вала (кривошипа), с–1.

Для рассматриваемого расчета ωвала кр = 2π∙n : 60 = 2π∙40 : 60 = 4,2 с–1. Рез пакета бумаги при движении ножа из верхнего крайнего положения в нижнее и обратно совершается в первой половине цикла, поэтому период работы под нагрузкой составляет примерно φрез = 140°…90° угла поворота кривошипа в зависимости от высоты стопы (первая цифра соответствует максимальной высоте стопы). Считаем, что принятая в примере высота стопы h максимальна, тогда – φрез = 140°, что соответствует 2,44 рад. Тогда фактическое время резания

tф рез = φрез / ωвала кр = 2,44 : 4,2 = 0,58 с,

а средняя мощность резания составит

Ррез = Арез : tф рез : ηмех = 9503,52 : 0,58 : 0,98 = 16719,77 Вт

Неравномерность затрат мощности за различные промежутки угла поворота кривошипа создает неравномерность угловой скорости его вала. В момент реза скорость вала уменьшается, а после его окончания восстанавливается до прежнего значения. Для уменьшения неравномерности вращения вала кривошипа применяют маховик, который накапливает энергию между рабочими циклами и отдает ее механизму ножа для выполнения операции резания. Такой прием позволяет уменьшить потребляемую приводным двигателем мощность. Маховик обычно устанавливается на высокоскоростном валу машины, на который поступает момент от вала двигателя после первой кинематической передачи, например, после клиноременной передачи соединяющей вал двигателя с маховиком. Весо-габаритные размеры маховика определяются тем объемом избыточной энергии, которую следовало бы передать механизму ножа в момент совершения реза для получения необходимой неравномерности вращения вала машины. Конструктивное исполнение маховика определяет его момент инерции, который вычисляется по формуле,

Jмах = Аизб : δ : ω2ср,

где    δ = (ωmax – ωmin) : ωср – коэффициент неравномерности вращения вала механизма;

          ωср – средняя скорость вращения вала механизма, с–1.

Избыточная энергия вычисляется по формуле

где    kзап = 1,4…1,5 – коэффициент запаса, учитывающий количество неучтенной работы механизма ножа (подъем ножа, трение в направляющих, силы инерции и т. п.) по отношению к работе сил резания;

          ηном – КПД механизма привода ножа.

Таким образом, необходимая от электродвигателя энергия резания составит

Аэн дв = Арез – Аизб = 9052,52 – 4213,13 = 4840,39 Вт

Мощность резания от этой энергии на кривошипе будет равна

Ррез дв = Аэн дв / tф рез / ηмех = 4840,39 /0,58/0,98 = 8515,82 Вт

Эта мощность создает на кривошипе момент равный

Мкрив = Ррез : ωвала кр = 8515,82 : 4,2 = 2027,58 Н∙м

Выберем для привода механизма ножа асинхронный электродвигатель с синхронной частотой вращения магнитного поля  ω0 = 157 с–1 и примерным скольжением  sном = 4%. Тогда номинальная скорость составит

ωном = ω0 (1 – sном) = 157∙(1 – 0,04) = 150,72 с–1

Исходя из соотношения скоростей вала двигателя и вала кривошипа, определим передаточное отношение кинематической передачи между ними

i = 150,72 : 4,2 = 35,88

Однако в действительности момент от электродвигателя передается кривошипу через клиноременную передачу с iкл = 2,5 (например, для ОРМ модели БР-82) и зубчатую кинематическую передачи с iзуб = 14,35. Таким образом, общее передаточное соотношение равно

i = iкл ∙ iзуб = 2,5 ∙14,35 = 35,875

 Приведенный момент сопротивления на кривошипе к валу двигателя определиться

Мпр крив = Мкрив : i : ηпер = 2027,58 : 35,875 : 0,96 = 58, 873 Н∙м.

Расчетная мощность выбираемого двигателя составит

Ррасч = Мпр крив ∙ ωном = 58,873 ∙ 150,72 = 8873,34 Вт

По справочнику выбирается электродвигатель типоразмера АИР132М4 со следующими номинальными данными: Рном = 11 кВт;  sном = 3,5 %; ηном = 87,5 %; cos φном = 0,87; λм = 2,2.

Определим геометрическую конфигурацию маховика, создающего рассчитанную избыточную энергию Аизб =  4213,13. Она выражается, прежде всего, в определении момента инерции маховика, причем, учитывая кратковременность действия возмущения (реза), полагаем δ = 1, а ωср = ωном : iкл = 150,72 : 2,5 = 60,288 рад/с – скорость вращения маховика 

Jмах = Аизб : δ: ω2ср = 4213,13: 1: 60,288 = 69,88 кг∙м2

Маховик-шкив представим в виде кольца со следующими параметрами:

r1 = 330 – внешний радиус шкива, мм (например, для модели БР-82),

r2 – внутренний радиус шкива, мм,

b = 115 – ширина шкива, мм,

γ =7,8 кг/дм3 – удельный вес материала, из которого сделан шкив (сталь).

Момент инерции маховика-шкива по конструктивным параметрам определяется выражением

Jмах = K∙mm∙(r12 – r22) : 2,

где    mm = π∙γ∙b∙(r12 – r22) – масса шкива;

K = 1,15 – коэффициент учитывающий конструктивную конфигурацию маховика-шкива.

Рассуждая аналогичным образом, можно по вариантам конструктивного исполнения маховика-шкива определять избыточную энергию, которая входит в энергию работы реза и может заменить часть мощности электродвигателя, уменьшая ее выбранное значение.
