5.4.1. Пример расчёта привода механизма подавателя

Пример рассчитывается для перемещения стопы бумаги с размерами (длина L-ширинаB-толщина Н) 1000-800-200 мм. Масса подавателя, перемещающая стопу, mпод = 110 кг. Механизм подавателя с ходовым винтом, расположенным под столом машины и обладающим ηпод = 0,92, а также с ползуном трапециидального вида, имеющим угол α = 60º. Ходовой винт изготовлен со следующими параметрами: средний радиус ходового винта rср = 16 мм, шаг ходового винта tх в = 6 мм (учитывая необходимость миллиметрового перемещения самого подавателя), ходовой винт имеет метрическую резьбу, у которой угол ее профиля φ = 60º, а коэффициент трения скольжения в паре винт-ползун  f3  = 0,15.

Определим массу стопы бумаги mс, которая вычисляется по формуле,

mс = L ·B ·Н· γ = 1,0·0,8 ·0,2· 1,0 = 0,16 т,

здесь  L, B, Н – длина, ширина и высота стопы, м;

           γ = 1,0 т/м3.

Тяговое усилие Fтяг, которое требуется приложить к подавателю, определится:

Fтяг = FΣ ин + G2 · f2 : sin α = 336 + 110 · 9,81 · 0,2 : 0,87 = 523,76 Н.

Суммарная сила инерционных масс подавателя со стопою

FΣ ин = Fин ст + Fин под + Fин пол = (mин ст + mин под + mин пол) ∙ апод = (160 + 110 + 10) · 1,2 = 336 Н,

где    апод – ускорение подавателя в период начала движения, которое, учитывая кратковременность этого периода (около 0,1 с), можно определить приблизительно, полагая, что интенсивность нарастания скорости в этот период происходит равномерно:

aпод = (νпод – ν0) : tразг = νпод  : tразг = 0,12 : 0,1 = 1,2 м/с2,

где    νпод – скорость подавателя при перемещении стопы, которая у большинства ОРМ имеет значение 0,12…0,16 м/с, для при мера примем νпод = 0,12 м/с;

         tразг – время разгона подавателя из состояния покоя (ν0 = 0) до νпод, для примера примем tразг  = 1 с.

Рассчитаем усилие, которое необходимо создать в начале движения, по формуле:

Fтрог = (mс + mпод) · g · m0 + Sпр · апр,

где    m0 = 0,25 – коэффициент трения в направляющих при трогании с места;

         Sпр – площадь прилипания стопы бумаги;

         апр = 0,3...0,5 Н/см2  – коэффициент, учитывающий свойство прилипания.

Современные резальные машины оснащены «воздушными подушками» как рабочего, так и приемного столов, поэтому будем считать, что указанный коэффициент трения отсутствует.

Тогда

Fтрог = (mс + mпод) · g · m0 = (160 + 110) · 9,81 · 0,25 = 662,175 Н.

Определяем  суммарную силу трения  FΣ тр механизма 

FΣ тр = Fтр с + Fтр под + Fтр пол =  mс · g · f1 + ψ∙ mпод∙ g ∙ k + [ mпол + (1 – ψ) ∙ mпод] ∙g ∙ f2 =

= 160 · 9,81 · 0,0 + 0,6 ∙ 110,0 ∙ 9,81 ∙ 0,1  + (10,0 + (1 – 0,6) ∙ 110,0) ∙ 9,81 ∙ 0,2 = 0,0 + 129,5 + 106,0 = 613,482 Н.

При движении в механизме будет действовать  усилие

Fус = Fтяг  + FΣ тр + FΣ ин = 523,76 + 613,482 + 336 = 1473,242 Н.

Момент статического сопротивления на ходовом винте

Mх в = Fус ∙ rср ∙ tg (α + β)

где    Fус  – осевое усилие, действующее на винт, Н;

         rср – средний радиус ходового винта, мм;

         α = arctg (tх в: π : rср)  – угол подъема винтовой линии;

         tх в – шаг ходового винта, мм; для примера зададимся  tх в = 6 мм, учитывая необходимость миллиметрового перемещения самого подавателя;

         β = arctg (f3  : (cos (φ:2))  – приведенный  угол трении в паре винт-гайка.

Здесь φ – угол профиля резьбы (при метрической резьбе φ = 60º ), а f3 = 0,1…0,2 – коэффициент трения скольжения.

Вычислим α и β

α = arctg (tх в : π : rср) = arctg (6 : 3,14 : 16) = 6,81,

β = arctg (f3  : (cos (φ:2)) = arctg (0,2 : (cos (60:2)) = 13,0    

Mх в = Fус ∙ rср ∙ tg (α + β) = 1473,242 ∙ 0,016 ∙ tg (6,81+ 13,0) = 8,485 Н∙м

Расчетная мощность приводного двигателя механизма подавателя:

Pрасч = Mх в ∙ ωх в : ηпод, 

где    ωх в = 2∙π ∙ νпод : tх в – угловая скорость вращения ходового винта, с–1;

          ηпод – КПД механизма привода подавателя.

Угловая  скорость   вращения   ходового   винта   в  рад/с, приняв νпод  =  0,12 м/с, составит

ωх в = 2∙π ∙ νпод : tх в = 2∙3,14 ∙ 0,12 : 6 = 0,1256 рад/с

Тогда, учитывая, что ηпод = 0,92, получим

Pрасч = Mх в ∙ ωх в : ηпод = 8,485 ∙ 0,1256 : 0,92 = 1,158 Вт.

Момент статического сопротивления на ходовом винте с учетом усилий при трогании составит

Mх в =( Fус + Fтрог) ∙ rср ∙ tg (α + β) = (1473,242 + 662,175) ∙ 0,016 ∙  tg (6,81+ 13,0) = 2135, 417∙ 0,016  ∙ 0,36 = 12,3 Н∙м

Расчетная мощность приводного двигателя тогда составит

Pрасч = Mх в ∙ ωх в : ηпод =  12,3 ∙ 0,1256  : 0,92 = 1,68 Вт.

Расчетная мощность приводного двигателя с учетом КПД механической передачи (ηпер = 0,96) от вала электродвигателя к ходовому винту будет равна

Pрасч дв = Pрасч : ηпер = 1,68 : 0,96 =  1,75 Вт.
